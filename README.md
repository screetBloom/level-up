# 我的前端成长之路和对未来的探索
在本仓库里，我计划记录一些自己的过去和未来的前端成长的历程

出身
---
- 网络工程出身，在学校时打交道最多的是交换机、路由器、服务器
- 学校是二本。在学习阶段，有过一些服务器虚拟化的实践，搭建过一个简易云平台；杂七杂八的学过浅显的java、安卓
- 真正开始学前端是大三，来自于一次学校组织的实训

前端经历
---
- 和大多数同学一样参加过校招；以前也没什么资源，不认识什么在大厂的师兄，都是笔试、多轮面试
- 实习是在"同程艺龙"
- 目前就职在上海的美团点评

成长之路的文章分享
---
前端真正启蒙是在同程艺龙火车票部门实习的那段时间，真正的稍微的打开了视野，了解到了一些非"画页面"的东西，最关键的是认识到了几个技术上对我有很大影响的人；以前一直不懂为什么他们代码要这么写，为什么可以写的比我好，思路哪来的
大家也知道，问这种问题是没有用的，没经历过、好好研究过，说再多也没用；那个时候一直维持到我去美团，我渡过了一段迷茫期，不知道前端该怎么学，大量的看知乎前端技术讨论、segmentfault、阮一峰、vue源代码分析，但是因为知识储备的问题，看起来有点懂，其实根本不知道如何运用、这些方案都是为了解决什么问题       那个时候我就一直模仿我们组的一个大牛，他的任何代码我都看，每次分享肯定去，听不懂先听着，自己后续再去看源代码，可以说整个同程的前端除了他没有人比我更清楚他写的代码；看得多了后来就学到了一些东西，一直影响我到现在，我是非常感激的
